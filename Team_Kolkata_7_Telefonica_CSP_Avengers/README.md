# Please go through the below instructions for accessing the TCS AI Friday LAB Resources.

### If you are allocated in cubicle number 2-60 (Hackathon Systems) then please follow the below instructions
    1. Please open the Network Shared Path "genailab-maas-hackathon".
    2. Navigate to "\\Hackathon\\Kolkata\\Gitanjali_Park\\Kolkata_AI_Friday_19-09-25_Master\\"
    3. You will find sub-folders with individual respective teams in the following format: "Team_{team_alias}_{team_name}"
    4. Please access only your respective Team's folder.

    5. Inside your respective team's folder, you will find the following:
        a. A Folder with the name "Test_Project-Sample_Code" - This folder contains the following:
            1. Sample codes to test your LLM model "llm.py"
            2. Sample codes to test the RAG functionality "vector.py" using Chroma DB and streamlit.
            3. TikToken_Cache folder and Chroma_Index folder. You will need these 2 folders to bypass the SSL verification process during embedding. You will need to specify the path of your "Tiktoken_Cache" directory in "vector.py"
            4. "requirements.txt" containing the required dependency python modules to run the above codes.

        b. An empty folder with the name "Solution" - You will need to create a .zip file of your entire solution and place only the zip file in this folder. Do not copy the ".env" or any other critical information in this zip file

        c. An empty folder "Presentation" - You will need to place your presentation file here for your solution.

        d. An empty folder "Recording" - You will need to place your screen recorded .avi file here.

        d. A ZIP file with the name "Screen_Recorder.zip".
            1. You will need to copy this to your "C:\Projects\" directory, after your solution and presentation is ready. 
            2. Unzip the file
            3. Open the extracted folder in VS Code
            4. Install the python dependency modules from the "requirements.txt" file.
            5. Execute the "recorder.py" file to start the recording.
            6. Walkthrough the entire PPT deck you have created with the recording on.
            7. Walkthrough your entire solution with the recording on.
            8. Stop the recording from your terminal by terminating the execution of "recorder.py"
            9. A recorded file with extension .avi will be generated in the same directory of "recorder.py" file.
            10. Copy the generated ".avi" recorded file to the "Recording" folder as specified in point d.

        e. You will also find the "AI Friday Participant Handbook.pdf"
    
    6. Please develop your solution in your local system "C:\Projects\<Your Solution Folder>"
